## Known limitations

- Hash-vector retrieval is the reproducible reference baseline, not the hosted semantic-quality ceiling.
- The evidence-extractive route intentionally returns source text rather than polished generated prose.
- Hosted Cloudflare and Claude evaluation requires credentials not stored in the repository.
- Independent human review cannot be truthfully replaced by a second automated pass from the same agent.
- No public widget or production deployment is included.
